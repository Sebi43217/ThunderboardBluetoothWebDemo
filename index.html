Thunderboard Demo

On this website you can use a Bluetooth-Client demonstration for the Silicon Labs Thunderboard type:SLTB010A.

This has been created with a custom Bluetooth library using the Web Bluetooth API. The website supports the same functions as the Thunderboard smartphone app. The 3D animation is created with the THREE 3D javascript library.
